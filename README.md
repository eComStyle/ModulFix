eComStyle.de::ModulFix
==========================
Hilft, wenn Module nicht installiert werden koennen.
  
#### Composer name
`ecs/modulfix


#### Shopversionen
`OXID eShop CE/PE 6`


#### Installation Produktiv-Umgebung
`composer require ecs/modulfix --update-no-dev`


#### Installation Dev-Umgebung
`composer require ecs/modulfix`
